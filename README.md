
# ahmad-portfolio-cosmic-wave

A polished interactive 6-page React portfolio using the 'Cosmic Wave' color palette.

## Quick start

1. Extract the ZIP and open terminal inside the project folder
2. Install dependencies:
   ```bash
   npm install
   ```
   If tailwind CSS shows issues, run:
   ```bash
   npm uninstall tailwindcss
   npm install -D tailwindcss@latest postcss autoprefixer
   npx tailwindcss init -p
   ```
3. Run development server:
   ```bash
   npm start
   ```
